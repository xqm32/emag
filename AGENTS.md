# AGENTS.md

## 项目信息

- **项目名称**: Osmos (WebGL2 细胞吞噬游戏)
- **主要文件**: `index.html`

## 开发约定

### Git 操作

- ✅ **可以自动执行**: `git add` 和 `git commit`
- ❌ **不要自动执行**: `git push`
- **Push 前**: 必须询问用户确认

### 项目特点

- 单文件 HTML 游戏，包含 WebGL2 渲染和物理逻辑
- 已实现功能：
  - 7 阶段渐进难度系统
  - 肉鸽元素（每30秒选择强化效果）
  - 15种可叠加的 buff 效果
  - 粒子喷射视觉效果

## 用户偏好

- 提交信息要清晰描述改动内容
- 视觉调整需要反复确认（如粒子效果）
- 重要操作前需要询问确认
